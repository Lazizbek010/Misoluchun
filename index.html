Animation

  • animation-name: Animatsiyani nomi;
  • animation-timing-function: Animatsiyani tezligi (Linear, ease, cubic);
  • animation-duration: Animatsiyani davomiyligi;
  • animation-delay: Animatsiya qancha vaqt kutib turib keyin boshlanishi;
  • animation-fill-mode: Forwards Backwords Both Animatsiya 0% da 100% qolib ketishi;
  • animation-iteration-count: Animatsiyani soni (infinite = cheksiz);
  • animation-direction: Animatsiyani yonalishi (Alternate = borib urilib qaytib kelishi);
  • animation-play-state: Running Paused Animatsiyani tohtab qolishi yoki harakat qilishi;

Position

  • Static Elementni default holati
  • Relative Element boshqa elementlarga halaqit bermasdan surilishi
  • Absolute oziga Relative ota qidiradi va uni maydoni boylab harakat qiladi
  • Fixed Ekranga yopishib qoladi va display xususiyati inline boladi
  • Sticky Otasini ichida qachonki topdan berilgan qiymatgacha static boladi va osha nuqtada fixedga ozgaradi qachonki otasini maydoni tugagunicha
